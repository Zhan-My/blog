# observable

# observable的实现

TIP

让一个对象可响应。Vue 内部会用它来处理 data 函数返回的对象。

返回的对象可以直接用于渲染函数和计算属性内，并且会在发生变更时触发相应的更新。也可以作为最小化的跨组件状态存储器，用于简单的场景。

# 文件结构

|——store
   |——index.js

# index.js

import Vue from 'vue'

Vue.observable({
  token: '',
  list: [],
  count: 0
  // ...也可以模块化
})

export default store;
